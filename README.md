# Asenkron Sayma (Async Counting)

Bu basit Python betiği, asyncio modülünü kullanarak asenkron programlama kavramını örneklemektedir. Kod, asenkron bir şekilde "Bir" ve "İki" kelimelerini ekrana yazdırır. asyncio.sleep() fonksiyonu aracılığıyla "Bir" kelimesini yazdırdıktan sonra bir saniye bekler ve ardından "İki" kelimesini yazdırır. 

## Kullanım

Betiği çalıştırmak için, terminal veya komut istemcisinde betik dosyasının bulunduğu dizine gidin ve aşağıdaki komutu çalıştırın:

```
python <dosya_adı>.py
```

## Gereksinimler

Bu betik, Python 3.7 veya daha yeni bir sürümü gerektirir.
